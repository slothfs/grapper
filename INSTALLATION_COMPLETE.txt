================================================================================
                          TROUBLESHOOTING
================================================================================

IF YOU SEE "Hook Tip not found" ERROR:
  → Check that Chain is instanced as child of Player
  → Verify the node is named "Tip", not "HookTip"
  → Check Scene tree structure matches documentation

IF HOOK NOT VISIBLE WHEN SHOT:
  → Verify Tip node has Visible=false in Inspector
  → Check Hook sprite texture is set
  → Verify collision isn't blocking visibility

IF PLAYER NOT MOVING TO HOOK:
  → Check Player has collision_layer=2, collision_mask=1
  → Verify hook_tip is being found (check console)
  → Ensure grapple_pull_speed is not 0

IF CAMERA NOT FOLLOWING MOUSE:
  → Verify Camera2D script is attached
  → Check follow_strength is not 0
  → Make sure Camera2D is child of Player

IF ROPE DOESN'T STRETCH:
  → Verify Links sprite has texture assigned
  → Check Region Enabled is true in Inspector
  → Verify Texture Repeat is enabled

================================================================================
                           TECHNICAL DETAILS
================================================================================

ARCHITECTURE
  • Player is a CharacterBody2D with CharacterBody2D physics
  • Hook (Tip) is a separate CharacterBody2D that moves independently
  • Chain rope is rendered as a stretched Sprite2D between player and hook
  • Camera follows player with mouse offset

COLLISION SYSTEM
  • Player collides with Ground (layer 1) via layer 2/mask 1
  • Hook collides with Ground via layer 4/mask 1
  • TileMap has collision mask 6 (binary 110 = layers 2 and 4)
  • This allows proper physics for both player and hook

GRAPPLE MECHANICS
  • When hook shoots, it moves with velocity until hitting collision
  • When active, player is pulled toward hook at grapple_pull_speed
  • Player can add sideways velocity with A/D keys while grappling
  • Partial gravity applied during swing for natural motion
  • Auto-releases when within 20 pixels of hook

CAMERA SYSTEM
  • Calculates offset between mouse and player position
  • Limits offset to max_distance_from_player
  • Lerps from current position to target position
  • Results in smooth, responsive camera that aids aiming
