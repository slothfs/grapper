# Grapper

# Description

Grapper is a physics-based puzzle and action game that blends soft-body mechanicswith a grappling hook system into something that is honestly kind of chaotic in 
the best way possible The game builds toward a boss fight against a giant sunand everything leading up to it keeps the pressure and the fun going at the same 
time

The player character is built using the Softbody 2D plugin in Godot 4.6.2, which gives it an internal bone structure and that wobbly, squishy feel when it moves 
around. The grappling mechanism was written entirely from scratch  no plugin for that part just pure code to get the hook feeling right.

# Why I Made This

This project was originally built as a submission for the ysws Beast  But beyond that it was the first time I sat down and made something based entirely on my own concept rather than following a tutorial or building on someone else idea

Part of the inspiration came from a monkey-themed game i used to play as a kid that had a swinging and grappling mechanic  always wanted to make something that had that same feel so when I finally had the time and the reason to do itthis is what came out of it

# Note

This is a prototype. The full version of the game is still in progress and will be released soon  you can all can use my scripts  it to get the same mechanisham in your game 

The scripts are inside the scripts folder. The assets are all in the assets folder. If you want to take things further visually, there are also shader scripts included in the shader section that you can drop in to make your game look more polished

# Controls
W A S D for movement
Right mouse click to shoot the grappling hook
E to release the grappling hook
Mouse movement to control the camera
